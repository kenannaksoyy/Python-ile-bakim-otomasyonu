# Python ile bakim otomasyonu
Tkinter kütüphanesi ile tasarlamış olduğum arayüz aracılığıyla çalışanların yapmış oldukları günlük iş başarılarılarına gore skorlama yapmaktadır.
Pandas ve openylx ile excelden dinamik veri alımı sağlanmaktadır.
Skorları takibi tarih formatına uygun olarak kaydedilmektedir.
